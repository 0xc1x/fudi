# Database Schema Changelog

Historial de cambios en el esquema de base de datos de Fudi.

## Formato

| Fecha | Versión | Autor | Cambio | Justificación | Migración | Impacto |
|-------|---------|-------|--------|---------------|-----------|---------|

## Entradas

---

> Cada entrada debe incluir:
> - **Fecha:** YYYY-MM-DD
> - **Versión:** Número de versión del schema
> - **Autor:** Agente o persona responsable
> - **Cambio:** Descripción del cambio realizado
> - **Justificación:** Por qué se hizo el cambio
> - **Migración:** Archivo de migración asociado
> - **Impacto:** Qué partes de la aplicación se ven afectadas
